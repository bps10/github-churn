Motivation

Open source development has produced millions of valuable products that are used by engineers around the world. The continued success of these projects, however, depends on a large and active community of developers willing to donate their time.

Consider the period between Jan-May 2016. During this time, there were 3.7 million users who had at least one open source event on GitHub. The probelm is that over the next 5 months 48% of those users made zero contributions. The goal of CodeChurn is to monitor and promote the health of the OS community.

CodeChurn helps you identify potential churners and suggests actions you can take to prevent it from happening.

Tech

The prediction engine behind CodeChurn was built using data from GitHub's User api and the publically available GitHub archive on Google BigQuery, which contains a record of all events on GitHub. For more information about the model and how it was implemented visit the CodeChurn github page .
